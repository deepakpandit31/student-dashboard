# Student Dashboard

A modern learning dashboard built using Next.js App Router, Supabase, Tailwind CSS, and Framer Motion.

## Features

- Responsive bento grid dashboard
- Animated sidebar and mobile navigation
- Dynamic course cards from Supabase
- Animated progress bars
- Loading skeletons
- Smooth Framer Motion animations
- Responsive dark mode UI

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase
- Lucide React

## Setup

Install dependencies:

npm install

Run development server:

npm run dev

Create a `.env.local` file using `.env.example`.

## Architecture

- Server Components used for data fetching
- Client Components used for animations and interactions
- Responsive layout using CSS Grid and Flexbox
- Framer Motion used for stagger animations and hover interactions
